# Test Images

Place local test images in this folder using the filenames below so the example commands work without modification.

Suggested layout based on the uploaded images:

```text
data/
├── arihant_selfie.jpg
├── arihant_mirror.jpg
├── group_indoor.jpg
├── group_night.jpg
├── far_stairs.jpg
└── far_outdoor.jpg
```

Recommended usage:

- `arihant_selfie.jpg`: primary registration image
- `arihant_mirror.jpg`: optional second registration image
- `group_indoor.jpg`: main recognition test
- `group_night.jpg`: harder recognition test
- `far_stairs.jpg`: likely too distant for reliable recognition
- `far_outdoor.jpg`: likely too distant for reliable recognition

Keep registration images to clear, close-up shots with one dominant face.
